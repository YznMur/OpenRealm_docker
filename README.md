# DDR-SLAM Docker
Docker for ```OpenREALM``` system

# Run the docker
The following command will open a docker terminal
```
./docker_build.sh
./docker_all_run.sh
./into.sh
```

Tree of OpenREALM workspace:
```

├── g2o
```
